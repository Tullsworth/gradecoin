Fingerprint

Definition

A fingerprint is a 256 bit 64 character hexadecimal user identifier for users. Fingerprints are used in defining users in transactions and blocks.

Fingerprint Generation

A user's finger print is generated via applying SHA256 sum of the user's public RSA key.
